# Neovim Configuration

Personal Neovim configuration with lazy.nvim plugin manager.

## Requirements

- Neovim 0.8+
- Git

## Installation

Clone the repository to your Neovim config directory:

```bash
git clone git@github.com:OOOytun/nvim.git ~/.config/nvim
```

Then start Neovim. Lazy.nvim will automatically load all plugins.

To manage plugins, use `:Lazy` command in Neovim.
